📚 Learn About Malicious URLs

🚨 Signs of Dangerous Links

  • Very long URL strings
  • Spelling mistakes in domain
  • Uses IP address instead of name
  • Contains words like verify/login/update
  • Shortened links hide destination
  • No HTTPS

🎯 Social Engineering Tricks

  • Urgency — “Act now!”
  • Fear — “Account blocked”
  • Reward — “You won prize”
  • Authority — Fake bank/government

🛡️ Safety Tips

  • Hover before clicking
  • Check domain carefully
  • Never enter OTP from unknown link
  • Use official websites only
